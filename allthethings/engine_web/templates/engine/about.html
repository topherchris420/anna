Vers3Dynamics Engineering Intelligence

An open-source, self-hostable, AI-powered search engine for engineering knowledge. It layers modern semantic retrieval and citation-first answers on top of a battle-tested Elasticsearch + Flask architecture.

How search works

Every query runs two retrievers in parallel: a BM25 lexical query and a dense-vector kNN query over locally generated sentence embeddings. Their rankings are merged with Reciprocal Rank Fusion, giving results that respect both exact keyword matches and semantic meaning. Answers are generated extractively by default (grounded in retrieved text with inline citations) and can use a local LLM when one is configured.

REST API

Everything the UI does is available as JSON under /api/v1: /search, /document/<id>, /summarize, /compare, and /collections. See /api/v1/health.

Extending it

Adding a new source is a single file that subclasses SourcePlugin (or the DocsCrawler base) and registers itself with @register. Drop it into engine/ingest/sources/ and it appears everywhere — no wiring required.

Built on the search architecture of Anna's Archive. The legacy book/paper search remains available at /legacy.
